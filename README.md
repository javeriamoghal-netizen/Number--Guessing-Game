# 🎯 Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python.
This project helps practice core programming concepts such as loops, conditionals, user input handling, and basic logic building.

---

## 📌 Project Description

The computer randomly generates a number between **1 and 100**.
The user keeps guessing until the correct number is found.

After each guess, the program gives feedback:

* 📉 **Too low**
* 📈 **Too high**
* ✅ **Correct**

The game continues until the user guesses the correct number.
